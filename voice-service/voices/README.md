# Stock voice reference clips

Drop clean 24kHz-ish mono speech samples here, named to match `ref` paths in
`../voices.seed.json` (e.g. `aria.wav`, `rex.wav`, `maya.wav`). 6–30s of clear
single-speaker speech is ideal. The service normalizes them on first use.

A built-in with a missing clip still appears in the picker but is greyed out
(`ready: false`) until its clip is added.
